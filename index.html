    Auto model

    This script let you show or hide a model just by naming your model with an Id then naming your model opener with the attribute model='id-of-model'. By add 'dp' to the model opener it can also close the model otherwise add close='id-of-model' to the close button.

    You can also add additional functions if adding 'js' to the model is not enough for you. All you have to do is to create a function with the name of the id. If the id has more than 2 words ('the-id-name') use first and last words and you can run a whole function onclick.

    With this Script you don't have to be adding numerous click event listeners.
